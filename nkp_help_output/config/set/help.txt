Modify Kommander configuration

Usage:
  nkp config set [command]

Available Commands:
  default-workspace Set the name of the default Workspace to use in all commands when none is provided

Flags:
  -h, --help   Help for set

Global Flags:
      --config string            Config file to use (default "/home/runner/.kommander/config")
      --context string           The name of the kubeconfig context to use
      --kubeconfig string        Path to the kubeconfig file to use for CLI requests.
      --request-timeout string   The length of time to wait before giving up on a single server request. Non-zero values should contain a corresponding time unit (e.g. 1s, 2m, 3h). A value of zero means don't timeout requests.
  -v, --verbose int              Output verbosity

Use "nkp config set [command] --help" for more information about a command.
